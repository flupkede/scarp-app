# Scarp

> Where to put the next mason jar.

A prioritization tool for landslide monitoring placement in Alaska.

Inspired by the work of independent geologist [Bretwood "Hig" Higman](https://groundtruthalaska.org/landslides)
and the National Geographic feature "Lessons of a landslide detective" (June 2026).
